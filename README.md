# Quran Web Application

A modern responsive Quran reading platform built with Next.js, Node.js, Express.js, and MongoDB.

## Features

- 114 Surah list
- Ayah details page
- Search ayah by translation
- Arabic font settings
- Font size customization
- Persistent settings via localStorage
- Responsive UI

## Tech Stack

Frontend: Next.js, Tailwind CSS
Backend: Node.js, Express.js, MongoDB
Deployment: Vercel 

## API Endpoints

GET /api/surahs
GET /api/surahs/:id
GET /api/search?q=keyword
